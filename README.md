# TMS Web (Demo v2)

- 参照モード（モーダル）を追加：操作列・新規ボタンを非表示、行クリックで呼び出し元へ返却
- トーストは `v-snackbar :timeout` により**自動で消える**
- 保存・削除・スキル保存など更新時は `v-overlay` とボタン `:loading` で「ぐるぐる」表示
- 参照成功は通知なし、異常終了は `ErrorDialog` で確認モーダル表示
- 薄青ホバー、シングルフィールドサーチ、折りたたみ、区分値の集中管理 など

## 起動
```bash
pnpm i  # npm i / yarn でも可
pnpm dev
```
